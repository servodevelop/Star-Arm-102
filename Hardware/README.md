# Hardware

本目录用于存放 Star Arm 102 的硬件相关资料，方便大家查阅零件清单、查看 CAD 图纸以及参考装配说明。



## 目录说明

- [parts/](./parts/): 各零件STEP 与 BOM
- [cad/](./cad/README.md): CAD工程图纸
- [assembly/](./assembly/README.md): 装配说明与装配参考资料
